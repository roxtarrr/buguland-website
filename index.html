Requisitos para pasar a Buguland 5 Nukeland:

El objetivo del paquete de mods es completar la línea de misiones "Derechos de Presumir": Fabricar uno de cada artículo creativo fabricable.

- En cada nivel de Tecnología y Magia tendrás la opción entre varios mods diferentes para progresar, logrando finalmente los materiales de nivel superior requeridos para el final del juego.

- Actualmente hay más de 650 misiones que te guían a través de la progresión. Puedes ver la mayoría de ellas en la pestaña de Imágenes de Enigmatica 2: Expert.

- Tendrás la oportunidad de explorar mundos personalizados.

- Menos de 20 muertes entre todos los jugadores. Muertes totales: 0

- Pasarse Minecraft.



Info:

- Server ip: buguland4skyland.ddns.net

- Minecraft Version: 1.12.2

- Número de mods: 269

- Cada nuevo jugador aparece automáticamente en una isla recién creada cuando se une al servidor.

Para interactuar con las islas, puedes usar los siguientes comandos, que todos comienzan con /island

  • create (opcional int/string)[tipo] - El comando create crea una nueva isla en el mundo.
  • invite [jugador] - El comando invite pide a otro jugador que se una a tu isla.
  • join - El comando join te hará unirte a una isla si has sido invitado recientemente con el comando invite.
  • leave - El comando leave te hace dejar la isla en la que estás, ir al spawn y limpiar el inventario. Si eres la última persona en la isla, la isla no puede ser reclamada nuevamente.
  • home - El comando home teletransporta al jugador de regreso a su isla hogar, pero debe estar por defecto a 500 bloques de distancia.
  • spawn - El comando spawn teletransporta al jugador al spawn.
  • reset (opcional int/string)[tipo] - El comando reset limpia la isla, el inventario y comienza de nuevo como lo hace el comando create.
  • onechunk - El comando onechunk crea un borde mundial del tamaño de un chunk en el spawn y restablece el spawn. Este comando está deshabilitado por defecto y debe ser cambiado en la configuración.

Los jugadores pueden decidir si vivir todos en la misma isla o si cada uno en la suya, pero la decisión ha de ser unánime
